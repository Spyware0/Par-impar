# Par ou Ímpar

Um simples jogo **Par ou Ímpar** feito em *Python*, ***espero que goste!***

> Dependências: python e/ou git

## Início

```shell script

apt-get update

apt-get upgrade -y

apt-get -y install python git

```

## Clonagem e acesso

```shell script

git clone https://github.com/gbLuis0/par-impar

cd par-impar

python main.py

```
