# Par ou Ímpar

Um simples jogo **Par ou Ímpar** feito em *Python*, ***espero que goste!***

## Início

```shell script

apt-get update

apt-get upgrade -y

apt-get -y install python3.10 git

```

## Clonagem e acesso

```shell script

git clone https://github.com/Spyware0/par-impar

cd par-impar

cd jogo

python3 index.py

```
